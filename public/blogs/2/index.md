
# 前言

如果你也想拥有一个我的世界服务器，你可能需要准备以下：
- 闲置电脑或游戏云服务器。
- 有公网访问的服务。

我有一台家里不用的笔记本电脑，拥有24G内存，不用来当我的世界服务器真是太浪费了。

正好可以办公室有免费的wifi。

# 开始

## 首先
你需要将笔记本的系统刷成Linux，Windows Server也行，但是感觉没有Linux好用。
我这里选择的是debian13的gui版本，毕竟命令行实在是太枯燥了。

## 安装java

运行我的世界需要jdk，服务端也是同理。

首先确定你要运行的我的世界版本。

下面是paper官方文档推荐表格。

![](/blogs/2/9b449baebe1f9ea5.png)

接下来就前往 
[paper下载页](https://papermc.io/downloads/paper)下载的我的世界服务端版本。
![](/blogs/2/b252cb54ec1512cb.png)

可以下载最新版本或在下面下载旧版本的服务端。

`
如果你想使用整合包的服务端，可以解压运行start.sh文件
`

在你喜欢的目录下新建一个文件夹，把下载好的paper.jar放进去。

例如：我在home目录下新建了一个文件夹Mc_Server。

这个时候右键这个新建文件夹的页面，选择打开终端。

输入su回车并输入密码获取root权限。

再输入
```
java -Xms4G -Xmx4G -jar paper.jar --nogui
```
其中4G为你的内存分配策略，尽量选择内存的90%。

比如我的电脑是24G内存。
```
java -Xms20G -Xmx20G -jar paper.jar --nogui
```
剩余的4G是给Linux与Gui留下的。

还有其中的~~paper.jar~~要替换为你实际下载的文件名称。

可以使用TAP按键自动补全

回车运行会发现需要同意EULA协议,在paper.jar同级目录下找到txt文件，将其中的flase改成true。

再次运行上面的命令，这时候服务器就在端口25565上开放了。

在终端输入
```
op Lie_Jiu
```
会给予玩家管理员权限。
其中**Lie-Jiu**是我的昵称。
